# Contributing to meteor-quizes

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/meteor-quizes.git`
3. Install Meteor: `curl https://install.meteor.com/ | sh`
4. Run the app: `cd meteor-quizes && meteor`

## Submitting Changes

1. Create a branch: `git checkout -b my-feature`
2. Make your changes and commit with a clear message
3. Push and open a Pull Request against `main`

## Code Style

- Follow existing patterns in the codebase
- Keep components small and focused

## Reporting Bugs

Open an issue using the Bug Report template and include steps to reproduce.
